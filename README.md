# projetocroom

O Croom é um software dedicado a instituições de ensino que tem como objetivo automatizar o processo de gerenciamento de seus alunos dentro da sala de aula, disponibilizada para Web e Mobile. Embora ainda seja um projeto bastante novo, ele oferece vários recursos, como módulos de frequência, notas, atividades e muito mais!

O Croom é baseado em uma arquitetura muito extensível que permite que você se adapte às suas necessidades sem nenhum grande esforço.
